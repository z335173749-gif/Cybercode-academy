## 📖 About The Project
CyberCode Academy is a fully interactive web-based learning platform built with a cyberpunk / hacker aesthetic. It helps beginners learn web development in a fun and structured way using:

- Step-by-step lectures
- Coding examples
- Interactive quizzes
- Progress tracking
- Achievement system
- Certificate generation

---

## ✨ Features

- 🧠 HTML, CSS & JavaScript Courses
- 🧪 Interactive Quizzes & Tests
- 📊 Student Progress Dashboard
- 🏆 Achievement Badges System
- 🎓 Certificate Generator
- 🔍 Search Functionality
- 🌐 Responsive Design (Mobile + Desktop)
- 🟢 Hacker / Binary Rain Theme UI
- 💾 LocalStorage Progress Saving
